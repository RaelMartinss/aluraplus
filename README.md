# Alura Combo+ Landing Page

## Descrição

Este projeto é uma página de destino (landing page) estática desenvolvida para promover o Combo+ da Alura. O Combo+ oferece acesso combinado ao Alura+ (plataforma de cursos de tecnologia) e Alura Língua (plataforma de aprendizado de idiomas) por um preço único. A página apresenta informações sobre os benefícios, dispositivos compatíveis e opções de assinatura.

## Funcionalidades

- **Seção Principal**: Apresenta o título promocional, imagem do combo e botões para assinatura (Combo+ e Alura+ individual).
- **Seções Secundárias**: Explica os benefícios como assistir em qualquer dispositivo, baixar cursos e acesso a múltiplas áreas de conhecimento.
- **Lista de Dispositivos**: Mostra ícones e nomes dos dispositivos compatíveis (TV, Computador, Celular).
- **Rodapé**: Contém links para informações legais, política de privacidade e ajuda, além do logo da Alura.

## Como os Usuários Podem Utilizá-lo

Para visualizar a página, simplesmente abra o arquivo `index.html` em qualquer navegador web moderno. A página é responsiva e utiliza CSS para layout em grid e flexbox, garantindo uma boa experiência em diferentes tamanhos de tela.


Alternativamente, você pode acessar a versão hospedada online em [https://aluraplus-delta-eight.vercel.app/](https://aluraplus-delta-eight.vercel.app/).


## Onde os Usuários Podem Encontrar Ajuda

Para obter ajuda sobre o Combo+ ou os serviços da Alura, visite o site oficial da Alura em [www.alura.com.br](https://www.alura.com.br) ou acesse a seção de ajuda no rodapé da página.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página e conteúdo semântico.
- **CSS3**: Estilização, incluindo variáveis CSS para cores, layout responsivo com grid e flexbox, e fontes do Google Fonts (Inter).
- **Imagens**: Arquivos de imagem armazenados na pasta `img/` para ícones, logos e backgrounds.
